## CN2 GIA 和普通线路，差距有多大

很多人问：普通日本 VPS 也能用，为什么要多花这个钱？

这问题我以前也问过自己。

普通线路走的是公共互联网出口，高峰期共享带宽，遇到拥堵就是你的问题。CN2 GIA 是电信的精品网络，物理上就是另一条路，不和普通流量挤。

实际体验上的差距：

- 普通线路晚高峰延迟可能从 80ms 涨到 180ms，甚至更高
- CN2 GIA 同时段延迟波动通常在 ±15ms 以内
- 丢包率差距更明显，普通线路高峰期丢包 5-15% 很常见，GIA 基本维持在 1% 以下

做代理节点的话，这个差距直接影响使用体验。做业务的话，影响的是用户留存。
